Queerevent

Queerevent is a web application to find and share events within the LGBT+ community.

This is an open-sourced MVC app developed with Python in the backend using Flask with Flask SQLAlchemy among other extensions such as Wtforms and Flask Login. The Frontend is developed with vanilla JS. It also makes use of the Google Maps and the Google Geocoding APIs.

This application is not deployed yet but its functioning can be seen in this video.

Discerart

Discerart is a web application for finding art online teachers.

This app is developed with Vue.js making use of Vuex and the Vue Router for building the routes. It uses Firebase in the backend for managing permanent data and authentication.

To see its whole functionality login using the credentials example@email.com, password 'example'.
