Queerevent

Queerevent es una aplicación web donde compartir y encontrar eventos LGBT+.

Es una aplicación MVC con licencia open source desarrollada con Python en el backend usando Flask con Flask SQLAlchemy y otras extensiones como Wtforms y Flask Login. El Frotend está desarrollado con JS vainilla. También hace uso de las APIs de Google Maps y Google Geocoding.

Esta aplicación aún no está desplegada pero se puede ver su funcionamiento en el siguiente video.

Discerart

Discerart es una aplicación web para encontrar profesores de arte online.

Esta aplicación está desarrollada con Vue.js. Hace uso de Vuex para almacenar y manejar datos y el Vue Router para construir las rutas. Para los datos permanentes y la autenticación de usuarios usa Firebase en el backend.

Para ver sus funcionalidades al completo se puede acceder con las credenciales example@email.com, contraseña 'example'.

90s Music Memory Game

Este es un juego de memoria responsive con una UI que simula un equipo de música de los 90.

Cada celda es un disco que me gusta, y cada vez que aciertas suena una canción del disco correspondiente.

Está desarrollado con vanilla Javascript y CSS y desplegado con Github Pages.
